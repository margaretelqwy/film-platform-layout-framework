# film-platform-layout-framework
该项目用于整理影视类网站平台布局框架信息，对首页导航层级划分、分类标签构成以及播放页信息区块安排进行系统归档。通过文字记录建立可持续更新的资料库，便于对比不同影视平台在布局规划方面的差异。仓库仅保存公开页面信息。
